## 安装

1. **克隆仓库**:
    ```bash
    git clone https://github.com/HarleyQiu/crawler-visa.git
    cd crawler-visa
    ```

2. **安装依赖**:
    ```bash
    go mod tidy
    ```

3. **编译项目**:
    ```bash
    go build -o your-project
    ```
